Poradniki

Instalacja

Uruchamiamy konsolę w folderze z pobranym plikiem z działu Pobieranie i wpisujemy:

pip install fslib-0.1.0-py3-none-any.whl

Tworzymy plik ini

Tworzymy nowy plik Pythona w dowolnym folderze i otwieramy go w dowolnym edytorze.

Wpisujemy:

import fslib

Jest to import pakietu fslib

plik = fslib.files.INIFile('moj_plik_ini.ini')

Tutaj tworzymy obiekt klasy INIFile

plik.set_item('imie','Michal')

Teraz dodaliśmy do pliku moj_plik_ini.ini item o nazwie imie z wartością Michal

print(plik.get_item('imie'))

Tą linijką wyświetlamy na ekranie wartość itemu imie z pliku moj_plik_ini.ini

Rezultat:

-Uruchomienie pliku z kodem pythona-

Michal

Funkcja replace_sequence

Funkcja replace_sequence zamienia linijkę w pliku na inną.

-Plik pliczek.txt-----------

1

2

3

--------------------------------

Tworzymy nowy plik Pythona w dowolnym folderze i otwieramy go w dowolnym edytorze.

Wpisujemy:

import fslib

Jest to import pakietu fslib

fslib.files.replace_sequence('pliczek.txt','2','4')

-Plik pliczek.txt po uruchomienu skryptu-

1

4

3

Funkcja remove_chars

Tworzymy nowy plik Pythona w dowolnym folderze i otwieramy go w dowolnym edytorze.

Wpisujemy:

import fslib

Jest to import pakietu fslib

tekst = 'XCXZXEXSXCX'

print(fslib.strings.remove_chars(tekst,'X'))

Rezultat:

-Uruchomienie skryptu Pythona-

CZESC
